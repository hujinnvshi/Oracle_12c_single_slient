# 环境配置文件

本目录存放各个环境的 Oracle 静默安装配置文件。

## 目录结构

```
environments/
├── dev/               # 开发环境
├── test/              # 测试环境
└── prod/              # 生产环境
```

## 使用说明

每个环境目录下会包含以下配置文件：
- `db_install.rsp` - 数据库安装配置
- `dbca.rsp` - 数据库配置
- `netca.rsp` - 网络配置

## 创建新环境配置

从 `rsp_templates/12.1.0.2/` 复制模板文件到对应环境目录，然后修改环境相关参数。

## 环境差异

每个环境需要修改的关键参数：
- 数据库名称 (ORACLE_SID)
- 数据存储路径
- 内存配置
- 字符集设置
- 端口配置
